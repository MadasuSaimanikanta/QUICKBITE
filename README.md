# QUICKBITE

QUICKBITE is a full-stack food ordering platform built with the MERN stack. It allows customers to browse restaurants, place orders, and enables restaurant owners to manage menus and orders.

## Technologies

* React.js
* Node.js
* Express.js
* MongoDB
* Redis
* JavaScript
* JWT Authentication

## Features

* Customer, Restaurant Owner & Admin roles
* Restaurant & menu management
* Search & filtering
* Cart & order management
* Order history
* Secure authentication
* Redis-based token management

## Workflow

1. Users register or log in.
2. Customers browse restaurants and menus.
3. Items are added to the cart and orders are placed.
4. Restaurant owners manage menus and incoming orders.
5. Admins oversee platform operations.

## Installation

1. Clone the repository
2. Install dependecies: npm install
3. Add environment variables: MongoDB, Redis, JWT, Judge0 and Gemini API keys
4. Run development server: npm run dev
5. Open Live server
